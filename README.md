## Halaman-halaman website La Forchetta
Website ini terdiri dari beberapa halaman utama, yaitu:
1. Dashboard
2. Login
3. Hasil Search
4. Detail Makanan
5. Archive Makanan
6. Wishlist Makanan
7. Profil

1. Login
2. Profil
3. List makanan
4. Category makanan
5. Detail makanan
6. Review makanan
7. Archive makanan (berdasarkan like dan comment)
8. Wishtlist makanan
